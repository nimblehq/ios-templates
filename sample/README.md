## Getting Started

### Requirements

- Ruby `3.2+`
- Xcode `26+`
- Swift `6.1+`
- [mise](https://mise.jdx.dev/) - manages Ruby, Tuist, and xcbeautify versions via `.mise.toml`

### Install Dependencies

- `mise install`
- `bundle install`
- `bundle exec arkana` _(generate keys from `.env`)_

Open `Sample.xcworkspace` in Xcode and build.
