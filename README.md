# ios-template
Our optimized iOS template used in our projects using Xcode Templates

## Requirements

Xcode 12.0

## Usage

Add Nimble's templates by placing them in the folder `~/Library/Developer/Xcode/Templates/` by running this script

```sh
$ bash install.sh
```
