## Getting Started

### Requirements

- Ruby `3.2+`
- Xcode `26+`
- Swift `6.1+`
- Tuist (version managed via `.mise.toml`)

### Install Dependencies

- `bundle install`
- `bundle exec arkana` _(generate keys from `.env`)_
- `tuist generate`

Build with Xcode.
